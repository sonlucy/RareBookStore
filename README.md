# 절판서점
> 수요자 : 공급자 = n : 1 경매 방식과 달리, 수요자 : 공급자 = 1: n 형태로 공급자들 간의 가격 경쟁을 유도하여 
역경매를 통해 가장 좋은 조건을 제시한 판매자를 구매자가 선택하여 거래할 수 있는 구매자 중심 웹 서비스
<br><br>구매자: "이 상품 판매 해줘" -> 판매자: "이 조건으로 판매할게"
